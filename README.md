# セミナータイマーアプリ

このアプリは、セミナーやプレゼンテーション、勉強会などで時間を管理するためのシンプルなタイマーです。視覚的なプログレスリングで残り時間を一目で確認でき、時間管理をサポートします。

## 機能

-   **時間設定:** 分単位でタイマーの時間を設定できます。
-   **スタート/ストップ:** タイマーの開始と一時停止を切り替えることができます。
-   **リセット:** タイマーを初期設定に戻します。
-   **視覚的な進捗表示:** 円形のプログレスリングが残り時間を視覚的に表示し、直感的に時間の経過を把握できます。
-   **終了通知:** タイマーが0になるとアラートで通知します。

## 使い方

1.  **時間設定:** 「分」の入力欄に、タイマーを設定したい時間を入力します。
2.  **スタート:** 「スタート」ボタンをクリックするとタイマーが開始します。ボタンの表示が「ストップ」に変わります。
3.  **一時停止:** タイマー実行中に「ストップ」ボタンをクリックすると、タイマーが一時停止します。ボタンの表示が「スタート」に戻ります。
4.  **リセット:** 「リセット」ボタンをクリックすると、タイマーが初期設定に戻ります。
5.  **進捗確認:** 画面中央の大きな数字と、その周りのプログレスリングで残り時間を確認できます。
